# TripWings - מערכת ניהול טיולים ✈️🌍

**TripWings** היא פלטפורמה מתקדמת לניהול והזמנת חבילות טיולים (Travel Packages).  
המערכת מאפשרת למשתמשים לגלוש, לחפש ולהזמין חבילות טיולים ברחבי העולם, בעוד שמנהלי המערכת יכולים לנהל את כל ההיבטים של המערכת: חבילות, תמונות, הזמנות, תשלומים, הנחות, תגובות ועוד.

---

## ✅ תכונות עיקריות

### 👤 למשתמשים
- **גלריית חבילות טיול**: חיפוש וסינון מתקדם לפי יעד, מחיר ותאריכים
- **הזמנות**: הזמנת חבילות עם אפשרות לבחור מספר חדרים
- **עגלת קניות**: הוספת חבילות לעגלה וביצוע רכישה מרוכזת
- **תשלומים**: ביצוע תשלום (PayPal Sandbox) ומעקב אחר סטטוס תשלום
- **רשימת המתנה**: הצטרפות לרשימת המתנה כאשר אין זמינות
- **התראות אימייל**: תזכורות אוטומטיות לפני מועד הטיול
- **הנחות**: הצגת הנחות פעילות לתקופות מוגבלות
- **ביקורות/תגובות**: הוספת דירוגים ותגובות לחבילות
- **ארנק דיגיטלי**: ניהול יתרה ומעקב תנועות
- **לוח בקרה אישי**: צפייה בהזמנות, תשלומים וטיולים קרובים

### 🛠️ למנהלים (Admin)
- **ניהול חבילות טיול**: יצירה, עריכה ומחיקה
- **ניהול תמונות**: הוספה וניהול תמונות לחבילות
- **ניהול הנחות**: יצירה ועדכון הנחות לתקופות מוגבלות (עד שבוע)
- **ניהול משתמשים**: צפייה וניהול משתמשים במערכת
- **ניהול הזמנות ותשלומים**: מעקב אחר סטטוסים
- **ניהול תגובות**: אישור ומחיקה
- **סטטיסטיקות**: נתוני מערכת כלליים

## 🧰 טכנולוגיות

### Backend
- **ASP.NET Core 8.0 (MVC)**
- **Entity Framework Core**
- **SQL Server / LocalDB**
- **ASP.NET Core Identity** (Authentication & Roles)

### Frontend
- **Bootstrap 5**
- **Bootstrap Icons**
- **JavaScript**
- **Razor Views**

### שירותים נוספים
- **PayPal Sandbox Integration**
- **Email Notifications (SMTP Gmail App Password)**
- **QuestPDF** (יצירת PDF: קבלות / מסמכים)
- **Background Services** (משימות רקע אוטומטיות)
